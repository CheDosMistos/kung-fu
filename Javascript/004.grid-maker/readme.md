# Grid Maker

Creates a grid with **6x4 character squares** (including borders), specifying the rows and colums by parameters.

```Example
node app.js 3 4
****************
*    *    *    *
*    *    *    *
****************
*    *    *    *
*    *    *    *
****************
*    *    *    *
*    *    *    *
****************
*    *    *    *
*    *    *    *
****************
```
